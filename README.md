### Ticket & Queue Management System
 
> A Java-based simulation of real-world service operations, built as a practical exercise in data structures and backend logic.
 
[![Java](https://img.shields.io/badge/Built%20with-Java-007396?style=flat-square&logo=openjdk&logoColor=white)](https://github.com/fauricio9656/Proyecto-SC-304)
[![Repo](https://img.shields.io/badge/View%20on-GitHub-181717?style=flat-square&logo=github)](https://github.com/fauricio9656/Proyecto-SC-304)
 
**Features:**
- Queue management using custom data structures
- Multi-type service boxes: regular, fast, and priority
- Ticket assignment logic based on service type
- System state persistence
- Report generation
- External API consumption (real-time exchange rate)
```java
// Example: Ticket assignment by service type
Ticket ticket = new Ticket(ServiceType.PRIORITY);
Queue queue = QueueManager.getQueue(ticket.getType());
queue.enqueue(ticket);
```
